# Changelog

## [1.0.0] - 2025-01-01

### Added
- 🎵 Bundled **Faaah** meme sound (MP3)
- 🔍 Exit-code detection via VS Code Shell Integration API (≥ 1.93)
- 🔍 Terminal output pattern scanning as secondary detection layer
- ⚙️ Configurable error patterns, ignored exit codes, volume, and debounce interval
- 🔇 Toggle command (`Terminal Error Sound: Toggle On/Off`)
- 🔊 Test command (`Terminal Error Sound: Play Test Sound`)
- 🖥️ Native playback on macOS (`afplay`) and Windows (PowerShell WMPlayer)
